

# 🌌 COSMIC RAYSHOWER

### Interactive Visualization of Extensive Air Showers in High-Energy Astrophysics

> A research-oriented simulation tool demonstrating how ultra-high-energy cosmic particles generate cascading particle showers in Earth's atmosphere.

---



---

## 🌍 Overview

**COSMIC RAYSHOWER** is an interactive educational web application that bridges **theoretical physics and visualization**.

It demonstrates how a **single high-energy cosmic ray** entering Earth's atmosphere produces a **cascade of secondary particles**, known as an *extensive air shower (EAS)*.

The project combines:

* 📖 Conceptual explanations
* 📐 Mathematical modeling (Heitler–Matthews Model)
* 🎮 Interactive simulation
* 🔬 Real-world experimental context

---

## 🎯 Why This Matters

Cosmic rays reach energies up to **10²⁰ eV** — far beyond what current human-made accelerators like CERN can achieve.

Understanding them helps answer:

* What are the most energetic processes in the universe?
* Where do cosmic rays originate?
* How does matter behave at extreme energies?

### 📐 Mathematical Framework (Heitler–Matthews Model)

Core equations governing the simulation:

```
N(t) = 2^(t/λ)
```

```
E(t) = E₀ / 2^(t/λ)
```

```
Xₘₐₓ = λ ln(E₀ / E_c)
```

These describe:

* Particle multiplication
* Energy degradation
* Shower maximum depth
